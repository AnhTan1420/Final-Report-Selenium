## How to install 
### Install Folder 

```
 git clone https://github.com/AnhTan1420/Final-Report-Selenium.git
```

### Install requirements.txt

```
 pip3 install -r requirements.txt 
```

## How to run
### Activate .venv

```
 source .venv/bin/activate
```

### Run File

```
 python scripts.py
```
